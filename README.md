# S206-Lab
Códigos dos exercícios do laboratório da matéria de Qualidade de Software

## Como inicializar o cypress
Utilizando npx:
```bash
npx cypress open
```

Utilizando yarn:
```bash
yarn run cypress open
```

## Como executar os testes do cypress com gerador de relatório
Utilizando npx:
```bash
npx cypress
```

Utilizando yarn:
```bash
yarn run cypress
```
